Построили дом для престарелых

Жители Сардаяла построили дом для престарелых в центре деревни. Он рассчитан на 60 постояльцев. «Здесь есть с кемпобеседовать. Здесь очень хорошо, тепло, уютно. Работники очень добрые и дружелюбные. Для меня это не просто дом, а рай», — рассказала 95-летняя Татьяна Борисовна. В доме для престарелых комнаты напоминают родные дома постояльцев: те же обои, картины, та же мебель. Здесь есть всё для пожилых: водные процедуры, комната для вязания, массажный кабинет, лечебный центр и библиотека. Дом открыт для родственников круглосуточно.
